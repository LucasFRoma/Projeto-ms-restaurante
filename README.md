# Projeto-ms-restaurante
Aplicação desenvolvida em Spring Boot com JPA/Hibernate para gerenciamento de restaurantes e reservas.

Funcionalidades:
- CRUD completo para entidades Restaurante e Reserva
- Relacionamento 1:N (um restaurante → muitas reservas)
- Validações com Jakarta Bean Validation
- Banco de dados em memória H2 para testes rápidos
- Estrutura organizada em Controller, Service, Repository e DTOs
- Esse projeto foi criado para estudo e prática de persistência com JPA e modelagem de relacionamentos em aplicações REST.
